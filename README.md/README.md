# 📊 Student Performance Analysis

## 📌 Project Overview

This project analyzes student exam performance to identify the key factors that influence outcomes. The goal was to move from raw data to actionable insights using SQL, Excel, and Power BI.

---

## 🎯 Objective

To answer the question:

> **What factors most affect student exam performance?**

---

## 🗂️ Dataset

* Source: Kaggle
* Contains:

  * Math, Reading, Writing scores
  * Gender
  * Lunch type
  * Test preparation course
  * Parental level of education

---

## 🛠️ Tools & Technologies

* **SQL** → Data extraction and analysis
* **Excel** → Data cleaning and ranking
* **Power BI** → Visualization and dashboard creation

---

## 🔍 Methodology

1. Imported dataset into SQL for structured querying
2. Cleaned and explored data in Excel
3. Calculated average scores across different groups
4. Measured factor impact using performance gaps
5. Built an interactive Power BI dashboard

---

## 📊 Dashboard Features

* **Overview Page**

  * Overall average score (KPI)
  * Gender comparison
  * Lunch type comparison

* **Key Drivers Page**

  * Test preparation impact
  * Parental education impact

* **Interactive Filters (Slicers)**

  * Gender
  * Test preparation course

---

## 📈 Key Insights

* Gender has **minimal impact** on performance
* Lunch type shows a **moderate influence**
* Test preparation has a **strong positive effect**
* Parental education is the **strongest factor**

---

## 🧠 Conclusion

Student performance is more strongly influenced by preparation and socioeconomic background than by gender.
